# Stock Portfolio Tracker (Python)

## Overview
This project is a menu-driven stock portfolio tracker.
Users can add stocks using symbols like AAPL, view their portfolio,
and save investment data to a CSV file.

## Features
- Stock symbols (AAPL, TSLA, etc.)
- Menu-based interaction
- Portfolio summary
- CSV file export
- Input validation

## Project Structure
Stock_Portfolio_Tracker/
├── stock_tracker.py
└── portfolio.csv

## ▶ How to Run
Run the following command:
python stock_tracker.py

## How It Works
- Stock prices are stored in a dictionary
- User selects actions from menu
- Portfolio data is stored in memory
- Data can be exported to CSV

## Customization Guide
- Add stocks in stock_prices dictionary
- Change file format from CSV to TXT
- Add profit/loss calculation logic
