# Hangman Game (Python)

## Overview
This is a console-based Hangman game built using Python.
The game includes Easy and Hard difficulty levels and uses
real-world technology-related words.

## Features
- Easy and Hard difficulty levels
- Limited wrong attempts
- Input validation
- Console-based interaction
- Technology-related words

## Project Structure
Hangman_Game/
└── hangman.py

## ▶ How to Run
Make sure Python is installed, then run:
python hangman.py

## How It Works
- User selects difficulty level
- A random word is chosen
- Player guesses one letter at a time
- Game ends when the word is guessed or attempts run out

## Customization Guide
- Change words in easy_words or hard_words list
- Modify attempts to increase or decrease difficulty
- Add new difficulty levels easily
