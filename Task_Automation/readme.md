# Task Automation Script (Python)

## Overview
This project automates real-world repetitive tasks such as:
- Moving image files
- Extracting email addresses from text files

## Features
- Moves .jpg and .png files automatically
- Extracts email addresses using regex
- File and folder handling
- Fully automated process

## Project Structure
Task_Automation/
├── automation.py
├── source_files/
│   └── emails.txt
├── images_moved/
└── extracted_emails.txt

## ▶ How to Run
Execute the script using:
python automation.py

## How It Works
- Scans source folder for files
- Moves image files to another folder
- Reads text file and extracts email addresses

## Customization Guide
- Change file extensions in the script
- Modify source folder name
- Add logging for moved files
