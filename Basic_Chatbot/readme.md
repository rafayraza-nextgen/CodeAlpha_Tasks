# Basic Chatbot (Python)

## Overview
This is a simple rule-based chatbot built using Python.
It responds to greetings, questions, and farewells using predefined rules.

## Features
- Rule-based responses
- Keyword matching
- Continuous conversation loop
- Graceful exit using "bye"

## Project Structure
Basic_Chatbot/
└── chatbot.py

## ▶ How to Run
Run the chatbot using:
python chatbot.py

## How It Works
- User input is analyzed
- Keywords are matched using conditions
- Appropriate responses are returned
- Chat continues until user exits

## Customization Guide
- Add new responses in chatbot_response function
- Improve keyword matching
- Change chatbot name or messages
